# 笔记软件前端
vue + ts  
计划写个网页，开发windows、android、linux各平台的浏览器，以实现跨平台，大幅提升开发效率

## 运行
1. clone
2. 架好后端，现有对应[后端项目](https://github.com/russionbear/notes-app-backend)
3. 可以直接访问现成页面`http://whitebear.ml/note/local`
4. `npm install`
5. `npm run dev`
